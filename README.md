# Welcome to Pyrolert!

## What is this system about?
Pyrolert, is a system designed to categorize smoke and differentiate between safe and dangerous sources in indoor environments, particularly school laboratories and kitchens. The system integrates multiple sensors to measure the size of particulate matter, gas composition, and ambient temperature, while also counting the presence of occupants within the facility. A predefined threshold for smoke detection was implemented to determine whether the smoke originates from harmless activities, such as cooking or controlled scientific experiments, or from hazardous conditions like burning materials or large fires. In the event of detecting dangerous smoke, the Convolutional Neural Network (CNN) model was used by the camera to count the number of occupants present in the room to assist with evacuation planning. The device will then issue immediate real-time alerts to designated personnel, enabling a quick response to potential fire threats.
